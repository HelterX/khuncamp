## Animation System (Exhaustive)

### 1. Preloader Sequence (2.5 seconds total)
- **0–2s:** Backdrop blur animates from 30px → 0px; hero image translates Y from -75% → 0%; percentage counter counts 0 → 100%
- **2–2.5s:** Preloader layer fades out (opacity 1 → 0)

### 2. Pixel Grid Reveal
- 30×17 grid of squares animates in
- Each cell: opacity 0 → 1 over 0.5s
- Easing: `cubic-bezier(0.215, 0.61, 0.355, 1)` (deceleration / ease-out-cubic)
- Sequencing: distance-from-bottom priority + randomised + wave-effect timing variations

### 3. UAV / Drone Frame Animation
- 113 sequential PNG frames loaded for drone reveal
- Playback tied to scroll progress (not time)
- At frame 63: display state switches (drone "activates")
- Creates cinematic weapon-system reveal effect

### 4. Cursor Effects
- **Spotlight:** Radial gradient follows mouse cursor — 390px circular blur using `rgba(255,0,0)` at varying alpha, `background-attachment: fixed`
- **Trail particles:** Left behind cursor, fade over 0.7s ease-out, scale from 1 → 0.6
- **Dotted mask pattern:** 32px dot grid that reveals on hover over certain zones

### 5. Text Scramble Effect
- Random character substitution using pool: `X O # * @ ! & ? % A B R 6 0 $`
- Duration: ~0.2s per character
- Stagger: 0.01s per character interval
- Trigger: element enters 70% viewport visibility threshold

### 6. Marquee Text
- Horizontal scroll, base speed 0.6px per frame
- Reverses direction on scroll-up
- Speed dampening using lerp factor 0.1 (smooth deceleration)
- Provides visual continuity between sections

### 7. Crosshair Animation
- Two lines extend: width animates 0 → 50% over 1s
- Triggered when crosshair element is detected at screen centre
- Opacity transitions on detection
- Reinforces targeting/weapons-system metaphor

### 8. Blinking Status Indicator
- Red square `[■]` element
- Keyframe: `fadeBlink` — 0%: opacity 1, 50%: opacity 0, 100%: opacity 1
- Duration: 1s, infinite loop
- Used as active/alert status signal

### 9. Night Mode Toggle
- Switch-thumb element: `translateX(0%)` → `translateX(100%)` on toggle

### 10. Smooth Scroll
- Lenis smooth scroll library (desktop only, >991px)
- Creates "robotic / fluid" feel — slightly slower than native scroll

## Unique UI Patterns

### Coordinate Display
- Fixed position, top-left corner
- Shows `X: [value]  Y: [value]` updating live on mousemove
- Bounded to tracking-area element
- Evokes: targeting computer, military GPS interface

### Pixel Grid Overlay
- Full-screen coloured square grid
- `z-index: 1000`, `pointer-events: none`
- Opacity transitions triggered by scroll position
- Appears/disappears creating "scanning" or "detection" visual

### Glassmorphic Hover Buttons
- On hover: button text skews `skewX(-13deg)`
- Inner text counter-skews `skewX(+13deg)` to stay readable
- Background: `RGB(249,0,0)` → `RGB(20,24,30)` colour swap
- Feels like a physical switch being thrown

### Horizontal Video Label System
- Labels positioned left / centre / right of video frame
- Dynamic "shopping" button with icon wrapper (16px height)
- Evokes: military HUD / targeting overlay labels

### Sticky Section Scroll
- Merchandise section: video sticks to viewport while user scrolls through content
- Crosshair interaction activates as user passes through
